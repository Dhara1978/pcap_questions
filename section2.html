The Hierarchy of Exceptions

  • Python exceptions are organized into a hierarchy with BaseException at the top
  • Commonly used exceptions include Exception, ArithmeticError, LookupError, and more

    • BaseException

    |--SystemExit

    |--KeyboardInterrupt

    |__Exception

     |--StopIteration

     |--ArithmaticError

     |__ ...

raise, raise ex

  • Used to manually raise an exception

    • Example:

    raise ValueError("An error occurred")

    print(dir(math))

assert

  • Used to debug code by testing conditions

    • Example:

    assert x > 0, "x must be greater than 0"

except E as e

  • Catches an exception and assigns it to a variable

    • Example:

    try:

     # code that may cause an exception:

    except ValueError as e:

     print(e)

The args Property

  • The args property contains the arguments passed to the exception

    • Example:

    try:

     raise ValueError("Invalid value")

    except ValueError as e:

     print(e.args)
