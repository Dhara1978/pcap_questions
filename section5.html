Section 5: Miscellaneous (22%)

Build Complex Lists Using List Comprehension

  • List Comprehensions:
  • Syntax: [expression for item in iterable if condition]

    • Example:

    numbers = [1, 2, 3, 4, 5]

    squares = [x**2 for x in numbers]

    Embed Lambda Functions into the Code

    • Lambdas: Defining and Using Lambdas
      • Lambda functions are small anonymous functions defined with the lambda keyword
      • Syntax: lambda arguments: expression
      • Example: add = lambda x, y: x + y creates a function that adds two numbers
    • Self-Defined Functions Taking Lambdas as Arguments
      • Functions can take other functions as arguments, including lambda functions

        • Example:

        def apply_function(f, value):

         return f(value)


        print(apply_function(lambda x: x * 2, 5))  # Output: 10

    • Functions: map(), filter()
      • map(): Applies a function to all items in an iterable

        • Example:

        nums = [1,2,3,4]

        doubled = list(map(lambda x: x * 2, nums))  # Output: [2, 4, 6, 8]

      • filter(): Filters items in an iterable based on a function returning True or False

        • Example:

        nums = [1,2,3,4]

        doubled = list(filter(lambda x: x % 2, nums))  # Output: [2, 4]

Define and Use Closures

  • Closures: Meaning and Rationale
    • A closure is a function that remembers the values from its enclosing scope even if that scope has finished executing
    • Useful for data encapsulation and managing state across function calls
  • Defining and Using Closures

      Example:

      def make_multiplier(n):

       def multiplier(x):

        return x * n

       return multiplier


      doubler = make_multiplier(2)

      print(doubler(5))  # Output: 10

Understand Basic Input/Output Terminology

  • I/O Modes
    • Modes: 'r' (read), 'w' (write), 'a' (append), 'b' (binary), 't' (text)
  • Predefined Streams
    • Standard input (sys.stdin), output (sys.stdout), and error (sys.stderr)
  • Handles vs. Streams
    • Handles: References to open files
    • Streams: Data flows (e.g., input or output)
  • Text vs. Binary Modes
    • Text mode: Reads and writes strings
    • Binary mode: Reads and writes bytes

Perform Input/Output Operations

  • The open() Function
    • Opens a file and returns a file object

      • Example:

      file = open('example.txt','r')

  • The errno Variable and Its Values
    • errno: Standard error numbers in the errno module

      • Example:

      import errno

  • Functions: close(), .read(), .write(), .readline(), readlines()
    • close(): Closes the file
    • .read(): Reads the entire file
    • .write(): Writes data to the file
    • .readline(): Reads a single line
    • readlines(): Reads all lines and returns them as a list
  • Using bytearray as Input/Output Buffer
    • bytearray: Mutable sequence of bytes

      • Example:

      data = bytearray(5) # Creates a bytearray of size 5
