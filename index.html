Section 1: Modules and Packages (12%)

Import Variants in Python

import:

  • Used to import an entire module

    • Example:

    import math

    print(math.sqrt(16)) # Output: 4.0

from import:

  • Used to import specific functions or variables from a module

    • Example:

    from math import sqrt

    print(sqrt(16)) # Output: 4.0

import as:

  • Used to give a module or function an alias

    • Example:

    import math as m

    print(m.sqrt(16)) # Output: 4.0

import \*:

  • Imports everything from a module
  • Not recommended for larger modules to avoid name conflicts

    • Example:

    from math import *

    print(sqrt(16)) # Output: 4.0

Advanced Qualifying for Nested Modules

  • You can use from to import nested modules

    • Example:

    from package.subpackage import module

    # Accessing functions from the nested module

    module.function()

  • Alternatively, you can use import for more explicit imports

    import package.subpackage.module

    # Accessing functions from the nested module

    package.subpackage.module.function()

The dir() function

  • Used to list the names of the attributes and methods in a module
  • Provides an overview of what’s available within a module

    • Example:

    import math

    print(dir(math))

The sys.path Varaible

  • Lists the directories Python searches for modules
  • You can append to this list to add new directories for module searching

    • Example:

    import sys

    print(sys.path) # Output the list of paths


    # Adding a new path

    sys.path.append('/new/path/to/modules')
